# 版本更新记录

## v3.0.3 (2025-12-29) - 🎯 极简版

### ✂️ 移除功能
- **能量模式**: 移除微光能量算力通道
- **连接信号塔**: 移除所有连接信号塔按钮和链接
- **安全承诺提示**: 移除数据安全提示文字
- **实验室模式**: 移除实验室模式开关及相关警告
- **高级配置区**: 移除整个高级配置折叠面板
- **AI生成模式**: 移除算力通道选择，仅保留API Key输入
- **复制提示词**: 移除复制提示词按钮
- **AI生成按钮**: 移除一键生成配置按钮

### ✨ 新增功能
- **简历可编辑**: 简历输入框改为可编辑模式，支持用户自定义简历
- **自动保存**: 简历内容实时自动保存到本地存储

### 🎯 保留功能
- **API Key配置**: 保留DeepSeek API Key输入功能
- **简历画像**: 保留简历配置功能（现为可编辑）
- **HR过滤**: 保留不活跃HR过滤功能
- **自动沟通循环**: 核心自动化沟通功能（在页面浮窗）

### 🎨 改进
- 界面极简化，只保留最核心功能
- 移除所有充值和信号塔相关入口
- 移除所有高级技术配置，降低使用门槛
- 专注API Key模式，提供稳定可靠的体验
- 简历支持用户自由编辑和自定义

## v3.0.1 (2025-12-27) - 🧭 自动沟通循环修复

### ✨ 新增功能
- **新开沟通页**: 自动沟通时优先打开新标签页进行沟通
- **自动填充话术**: 新标签页自动填入打招呼话术

### 🔧 改进
- 发送按钮兼容 `.send-message` 且移除 `disable/disabled` 再点击

- 关闭逻辑仅针对沟通页/详情页，避免影响主列表页

## v3.0.0 (2025-12-27) - 🎯 个性化定制版

### ✨ 新增功能
- **预设简历模板**: 内置林承列的完整简历，自动填充
- **预设 API Key**: 内置 DeepSeek API Key，开箱即用
- **只读简历**: 简历内容锁定，防止误修改
- **优先沟通**: 分析职位后自动打开沟通窗口，立即发起对话
- **智能关闭**: 沟通完成后自动关闭浏览器页面（非浏览器窗口）

### 🔧 改进
- 简历输入框设为只读状态，灰色背景提示
- API Key 自动回填机制
- 新用户自动使用预设配置
- 职位分析完成后自动跳转至沟通 Tab
- 优化沟通流程：分析 → 打开沟通 → 关闭页面，完整闭环

### 🛠️ 技术细节
- 添加 `DEFAULT_RESUME` 常量 (popup.js:7-45)
- 添加 `DEFAULT_API_KEY` 常量 (popup.js:4)
- 优化 `loadSettings()` 自动填充逻辑 (popup.js:328-344)
- 优化 `saveSettings()` 空值保护 (popup.js:302-307)
- 添加只读样式 #resume[readonly] (style.css:531-536)
- 实现分析后自动打开沟通 Tab 功能
- 实现沟通完成后智能关闭页面功能
- 完整的职位分析 → 沟通 → 关闭流程
