# 版本更新记录

## v3.0.14 (2026-06-03) - ☁️ 云端日志与扫描简报版

### ✨ 新增
- **云端日志上传**: 浮窗新增“云端日志”配置入口，支持填写上传 URL 和 token。
- **服务器日志脚本**: 新增 `server/auto_apply_log_server.js`，可部署到云服务器并按 JSONL 追加保存完整日志。
- **失败补发队列**: 云端上传失败时保留少量待补发日志，不影响自动沟通循环。

### 🎯 优化
- **岗位简报简化**: 简报不再按 jobId 聚合，只导出每次扫描产生的 `checked` 记录。
- **后台上传链路**: 云端 POST 交给 background service worker 执行，降低页面 CORS / 混合内容拦截风险。
- **完整日志保留**: 明细日志和云端上传都保留完整字段，不裁剪薪资、探针、岗位描述等复盘信息。

---

## v3.0.13 (2026-06-03) - 🧾 岗位简报导出版

### ✨ 新增
- **岗位简报导出**: 新增“岗位简报”按钮，按 jobId 聚合，一岗一行。

### 🎯 优化

## v3.0.12 (2026-06-03) - 📝 岗位描述导出版

### ✨ 新增
- **岗位描述字段**: 自动沟通日志导出新增 description、descriptionSource、descriptionProbe。
- **职位描述邻近扫描**: 通过“职位描述/岗位描述/工作描述”标题定位正文，再向父容器和后续兄弟节点提取内容。
- **描述 DOM 复盘线索**: 导出日志记录描述来源和命中的 DOM 文本片段。

### 🎯 优化
- **描述提取兜底**: 标准 `.job-detail-body`、`.job-sec-text`、`.text-container` 失效时，使用标题邻近算法补救。

---

## v3.0.11 (2026-06-03) - 🔢 薪资混淆数字解码版

### ✨ 新增
- **Boss 字体混淆数字解码**: 将 ``-`` 自动转换为 `0`-`9`。
- **混淆薪资识别**: 支持从 `-元/天` 提取出 `200-220元/天`。

### 🎯 优化
- **日志可读性**: salaryProbe 中的薪资文本会输出解码后的数字，方便复盘。

---

## v3.0.10 (2026-06-03) - 🎯 标题邻近薪资版

### ✨ 新增
- **标题邻近扫描**: 通过岗位名称定位 DOM 节点，再向父容器查找同一区块里的薪资。
- **卡片父级兜底**: 当前卡片没有薪资时，继续扫描父级容器，适配薪资与标题分列的卡片结构。
- **全角横线兼容**: 支持 `200－250元/天`、`200–250元/天` 等薪资写法。

### 🎯 优化
- **失败线索增强**: 薪资仍为空时，salaryProbe 会导出当前卡片或父容器文本片段，不再静默空白。

---

## v3.0.9 (2026-06-03) - 🔎 薪资 DOM 探针版

### ✨ 新增
- **薪资 DOM 探针**: 自动扫描详情页和列表卡片中的 salary、job-salary、wage、pay 等候选元素。
- **列表卡片兜底**: 详情页薪资为空时，从当前岗位卡片文本中提取日薪/K薪资。
- **薪资复盘字段**: 导出日志新增 salarySource、salaryProbe，记录薪资来源和命中的 DOM 文本片段。

### 🎯 优化
- **K 薪资过滤更可靠**: 自动沟通前会先用详情页薪资，再用列表卡片薪资兜底判断。
- **无控制台复盘**: 即使 Boss 页面打开控制台会刷新，也能通过导出日志反查薪资 DOM 线索。

## v3.0.8 (2026-06-03) - 📤 自动沟通日志版

### ✨ 新增
- **自动沟通日志**: 记录 checked、skipped_filter、pending_opened、sent、failed、timeout 等关键动作。

## v3.0.7 (2026-06-03) - 🛡️ 自动沟通保护版
